Welcome to your first coding challenge!

The purpose of this challenge is to help ensure you have grasped the material we have covered so far in this course. Please follow the steps below as best you can. Don't be afraid to raise your hand and ask if you are stuck on something - I'm here to help!

Please create a new project containing the two following pages:

  1. An index.html page
  2. A contact.html page

Your pages should both include AT LEAST the following elements (bonus points if you use more!):

  1. HTML5 [nav] with navigation linking to the other page (if on the index page, should link to the contact page, and vice versa)
  2. An HTML5 [section] that contains placeholder text and a 200x200px placeholder image
  3. In your contact page, create a [form] element with two different types of [inputs]

Please add 3 of the following style rules to either (or both!) of your pages:

  1. A fixed nav bar with a background color that takes up 100% of the browser length
  2. display: flex to style the content on your page
  3. A box-shadow or text-shadow on one of your elements
  4. Text with a :hover or :active effect
  5. A fun font from google fonts, applied to any or all elements
  6. A CSS reset
*You must use at least (1) element selector, (1) class selector, and (1) id selector in your code.*

Happy coding!
