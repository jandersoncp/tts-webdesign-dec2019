We briefly covered animations in our previous class on Tuesday, but it's time to go in depth.

Browser prefixes - some older browsers need specific prefixes to understand the animation properties

  • -webkit-animation -----> CHROME
  • -moz-animation -----> FIREFOX
  • -o-animation -----> OPERA
  • -ms-animation -----> IE/EDGE

Keyframes

When you specify CSS styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times.

NOTE: To get an animation to work, you must bind the animation to an element.

You can also use percentages to define animation timing.

Other cool things you can do:

Delay an animation using animation-delay: Xs

Set how many times an animation should run using animation-iteration-count: X or infinite

Run in reverse direction or alternate cycles using animation-direction: revers, normal, alternate or alternate-reverse

Specify the speed curve of an animation using animation-timing-function: linear, ease, ease-in, ease-out, ease-in-out

Animation shorthand

Turns this:

  • animation-name: example;
  • animation-duration: 5s;
  • animation-timing-function: linear;
  • animation-delay: 2s;
  • animation-iteration-count: infinite;
  • animation-direction: alternate;

Into this:

  • animation: example 5s linear 2s infinite alternate

Browser prefixes - some older browsers need specific prefixes to understand the animation properties

  • -webkit-transition -----> CHROME
  • -moz-transition -----> FIREFOX
  • -o-transition -----> OPERA
  • -ms-transition -----> IE/EDGE

To create a transition effect, you must specify two things:

  • the CSS property you want to add an effect to
  • the duration of the effect
  • Note: If the duration part is not specified, the transition will have no effect, because the default value is 0.

Other cool things you can do:

Delay a transition using transition-delay: Xs

Specify the timing of a transition using transition-timing-function: linear, ease, ease-in, ease-out, ease-in-out

Transition shorthand

Turns this:

  • transition-property: width;
  • transition-duration: 5s;
  • transition-timing-function: linear;
  • transition-delay: 2s;

Into this:

  • transition: width 2s linear 1s;
