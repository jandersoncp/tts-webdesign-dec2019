CHALLENGE 3

Calculating gratutity is a repetitive task, so let's create a couple of functions that do the work for us.

  1. create a variable titled billAmount and store a random number (ie: 100)
  2. create a function titled gratuity()
  3. Gratuity should:
    1. multiply the value of billAmount by 20% (hint: use 0.2)
    2. return the value
  4. create a function titled totalWithGrat()
  5. totalWithGrat should:
    1. take in the amount as an argument
    2. call the gratutity function
    3. add that to the original bill amount
    4. return the total bill + gratuity
    5. log the total (with gratuity) to the console
  6. append new total to the following phrase: "your total including gratuity is:"
  7. Limitation: You can only invoke the totalWithGrat function when logging the result

DOM Events

Elements emit events based on user input. You can run code in response to them. Events include

Mouse events - click, mouseover, mouseout Keyboard events - keydown, keyup, etc Form events - submit, blur, focus, change, window events - load, hashchange, etc. touch events - touchstart, touchend, etc.
