Javascript review

Some quick facts:

  • javaScript runs on the client side AND the server side (but we will use it exclusively on the client side
  • Variables are declared with var. All statements in JS end with a semicolon.
  • When loading a script, always load before the closing body tag. If loaded in the head, it will block all resources on the page from loading until the javascript has loaded. This will slow down your browser tremedously.
