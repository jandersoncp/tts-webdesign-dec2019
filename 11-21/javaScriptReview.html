Functions

  • Functions are procedures that take arguments and return values.
  • Think about them like a sheet of paper with instructions on it.
  • A function has a name, an argument list, and a body.
  • Arguments can be named anything you want
  • Functions can complete a task OR return a value

The DOM

  • When a page is loaded in the browser, it creates a virtual copy of the page called the DOM (document object model)
  • Allows you to find elements (nodes) in the document
  • Edit, add, or remove nodes
  • Attach event handlers that respond to user input.
