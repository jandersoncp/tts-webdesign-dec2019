Practicing pushing up files to Github

git status - checks the status of the local repository and compares it to the remote repository git add [filename] - tracks the file in your local repository that has changed and adds it to the changelist git commit -m [message] - creates a commit with a message attached git push origin [branchname] - pushes your commit from your local repository to the remote repository in github
